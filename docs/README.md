# AI Huevos - Investor Presentation

Este directorio contiene la presentación unificada para inversores, accesible a través de GitHub Pages.

## Estructura

- `index.html` - Página principal de presentación para inversores
- `archetypes.html` - eBook completo con casos de estudio de Metro y Kikes

## Contenido

### Página Principal (`index.html`)
- **Hero Section**: Introducción a AI Huevos e infraestructura de agentes
- **Problema**: Los 4 desafíos críticos de operadores LatAm
- **Solución**: Enfoque freemium + phased deployment
- **Casos de Éxito**: Metro Hotels y Huevos Kikes (resumen)
- **Mercado**: Oportunidad TAM y métricas clave
- **Los 7 Patrones de Transformación**: Framework compartido

### eBook de Arquetipos (`archetypes.html`)
Análisis profundo de dos arquetipos operacionales en LatAm:

#### Metro: Operador Inmobiliario Hotelero
- Portfolio de 17 propiedades en 3 países
- Desafíos: data latency, silos operacionales, stack fragmentado
- Transformaciones: real-time dashboards, AI-driven pricing, cloud unificado
- Métricas no-negociables: occupancy, ADR, RevPAR, GOP, labor cost

#### Kikes: Productor Agroindustrial Integrado
- #1 productor de huevos en Colombia
- Desafíos: bioseguridad reactiva, trazabilidad parcial, datos fragmentados
- Transformaciones: IoT predictivo, blockchain trazable, AI commodity forecasting
- Métricas no-negociables: lay rate, mortalidad, feed conversion, yield

## Navegación

Desde la página principal, los usuarios pueden:
1. Ver resumen ejecutivo de la oportunidad
2. Explorar casos de éxito resumidos
3. Acceder al eBook completo con análisis detallado
4. Solicitar diagnostic gratuito

## Mantenimiento

Para actualizar el contenido:
1. Editar `index.html` para cambios en la presentación principal
2. Editar `archetypes.html` para cambios en casos de estudio
3. Commit y push a `main` (o `development` → PR → `main`)
4. GitHub Pages se actualiza automáticamente
