# Research - Investigación y Análisis

Repositorio de investigación de usuarios, análisis de mercado, y performance reports.

## Directorios

### `/user-personas/`
Personas de usuarios:
- Perfiles detallados
- Necesidades y pain points
- Comportamientos y motivaciones
- Journey maps

### `/market-analysis/`
Análisis de mercado:
- Competitive analysis
- Market trends
- Opportunity mapping
- Industry insights

### `/performance-reports/`
Reports de performance:
- Channel analytics
- Campaign results
- A/B testing results
- ROI analysis

## Metodología

### User Research
1. **Discovery**: Entrevistas, encuestas, observación
2. **Analysis**: Identificación de patrones y insights
3. **Synthesis**: Creación de personas y journey maps
4. **Validation**: Testing y refinamiento

### Market Analysis
1. **Research**: Recopilación de data de mercado
2. **Competitive**: Análisis de competidores
3. **Trends**: Identificación de tendencias
4. **Opportunities**: Mapping de oportunidades

## Comandos Claude

```bash
/sc:research        # Ejecutar análisis de research
/user-research      # Workflow de user research
/channel-report     # Generar performance report
```
