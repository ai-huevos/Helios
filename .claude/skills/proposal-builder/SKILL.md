# Proposal Builder Skill

Create professional proposals that win business and set clear expectations.

## Quick Start

| Document Type | Use When |
|---------------|----------|
| **Discovery Proposal** | First engagement, diagnostic scope |
| **Implementation Proposal** | Post-diagnostic, automation build |
| **Retainer Proposal** | Ongoing advisory relationship |
| **SOW** | Formal scope after verbal agreement |
| **Pricing Sheet** | Quick quote, add-on services |

## Proposal Structure (Universal)

### 1. Executive Summary (1 page)
```markdown
## Por Qué AI Huevos

[1 paragraph: their problem + our unique solution]

### Lo Que Proponemos
[2-3 bullet summary of deliverables]

### Inversión
[Total price, payment terms summary]

### Próximo Paso
[Specific call to action]
```

### 2. Understanding (Their World)
```markdown
## Entendemos Su Situación

[Reflect back their pain points - show you listened]

### Desafíos Actuales
- [Challenge 1]
- [Challenge 2]

### Lo Que Está en Juego
[Quantified cost of inaction]
```

### 3. Solution (Your Answer)
```markdown
## Nuestra Propuesta

### Fase 1: [Name]
**Duración:** X semanas
**Objetivo:** [outcome, not activities]

Entregables:
- [Deliverable 1]
- [Deliverable 2]

### Fase 2: [Name]
[repeat pattern]
```

### 4. Investment (Pricing)
```markdown
## Inversión

| Concepto | Precio |
|----------|--------|
| Fase 1: Discovery | $XX,XXX USD |
| Fase 2: Implementation | $XX,XXX USD |
| **Total** | **$XX,XXX USD** |

### Términos de Pago
- 50% al firmar
- 50% al entregar

### Qué Está Incluido
- [Inclusion 1]
- [Inclusion 2]

### Qué No Está Incluido
- [Exclusion 1]
```

### 5. Why Us (Credibility)
```markdown
## Por Qué AI Huevos

### Experiencia Relevante
- [Client/project 1]: [result]
- [Client/project 2]: [result]

### Nuestro Enfoque
"Arqueología Organizacional" — [brief methodology description]

### Equipo
[Key team members, relevant background]
```

### 6. Terms & Next Steps
```markdown
## Términos y Condiciones

### Validez
Esta propuesta es válida por 30 días.

### Confidencialidad
[Standard confidentiality clause]

## Próximos Pasos

1. Revisar y aprobar propuesta
2. Firmar acuerdo
3. Agendar kick-off
4. Iniciar [Phase 1]

---

**Para proceder:** Responda a este correo confirmando su aceptación.
```

## Pricing Frameworks

### Discovery/Diagnostic
```
Formula: (Interviews × $500) + (Analysis × $2,000) + Deliverables

Example (20 interviews):
- Interviews: 20 × $500 = $10,000
- Analysis: 3 weeks × $2,000 = $6,000
- Report + Presentation: $4,000
- Total: $20,000 USD
```

### Implementation
```
Formula: (Complexity × Base Rate) + Infrastructure + Support

Complexity tiers:
- Simple automation: $5,000-15,000
- Medium integration: $15,000-40,000
- Complex system: $40,000-100,000+
```

### Retainer
```
Monthly options:
- Advisory (4 hrs): $2,000/month
- Embedded (20 hrs): $8,000/month
- Fractional CTO (40 hrs): $15,000/month
```

## Quality Checklist

Before sending proposal:

- [ ] Client name spelled correctly
- [ ] Dates are realistic
- [ ] Pricing adds up correctly
- [ ] No track changes visible
- [ ] PDF exported (not editable Word)
- [ ] Follows AI Huevos brand (if visual)
- [ ] Clear next step with deadline
- [ ] Mobile-readable formatting

## Proposal Tactics

### Anchor High
- Present premium option first
- Make middle option the "obvious" choice
- Include basic option for price-sensitive

### Remove Friction
- One clear call to action
- Simple approval process
- Reply-to-accept when possible

### Create Urgency (Authentic)
- Limited availability
- Price valid for X days
- Project timing dependencies

### Handle Objections Preemptively
- Address common concerns in the doc
- Include "Why This Investment" section
- Show ROI calculation
