‚úÖ Quick Review Checklist

Use this checklist when reviewing design implementations:

Design System Compliance

  • All colors use CSS variables (no hardcoded hex)
  • Typography uses defined scale (no arbitrary sizes)
  • Spacing uses token system (--space-*)
  • Components follow naming conventions

Accessibility

  • WCAG AAA contrast ratios (21:1 for body text)
  • Keyboard navigable
  • Screen reader friendly (semantic HTML)
  • Respects prefers-reduced-motion

Brand Consistency

  • Logo used correctly (official SVG files)
  • Brand colors used consistently
  • Typography matches brand guidelines
  • Tone is bold, direct, confident
