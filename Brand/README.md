# Brand - Identidad de Marca

Directorio central para toda la identidad de marca de AI Huevos.

## Estructura

### `/visual-identity/`
Elementos visuales de la marca:
- **logo/** - Archivos de logo en diferentes formatos y variaciones
- **colors/** - Sistema de colores, paletas, tokens
- **typography/** - Tipografías, jerarquías, uso
- **assets/** - Recursos visuales adicionales

### `/voice-tone/`
Guías de voz y tono de marca:
- Personalidad de marca
- Estilo de comunicación
- Ejemplos de uso
- Do's and Don'ts

### `/messaging/`
Mensajes clave de marca:
- Value propositions
- Taglines
- Elevator pitches
- Key messages por audiencia

## Uso

Para cualquier creación de contenido o diseño, consultar primero estos guidelines para asegurar consistencia de marca.

## Comandos Claude

```bash
/brand-check    # Validar consistencia con brand guidelines
/sc:design      # Operaciones de diseño y marca
```
