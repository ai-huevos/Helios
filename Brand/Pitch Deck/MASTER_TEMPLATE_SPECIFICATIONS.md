## Template 1: Master - Title Slide

### Layout Structure
- **Frame Size**: 1920×1080px
- **Background**: #FFFFFF (White)
- **Auto Layout**: Vertical, Scale mode
- **Padding**: 120px all sides

### Content Hierarchy (Top to Bottom)
1. **Logo Container**
   - Width: 200px (auto height)
   - Horizontal alignment: Center
   - Spacing below: 64px

2. **Headline Text**
   - Font: Scandia Medium
   - Size: 100px
   - Color: #121212 (Black)
   - Letter spacing: -1%
   - Horizontal alignment: Center
   - Spacing below: 32px

3. **Tagline Text**
   - Font: Scandia Regular
   - Size: 24px
   - Color: #121212 (Black)
   - Letter spacing: -1%
   - Horizontal alignment: Center
   - Spacing below: 16px

4. **Subline Text** (Optional)
   - Font: Scandia Regular
   - Size: 18px
   - Color: #9B9B9B (Gray)
   - Letter spacing: 0%
   - Horizontal alignment: Center

### White Space
- Minimum 60% white space
- Vertical centering with generous padding

### Auto Layout Settings
- Direction: Vertical
- Alignment: Center
- Padding: 120px
- Gap: Auto (64px, 32px, 16px as specified)
- Resizing: Scale

## Implementation Checklist

For each template:
- [ ] Create frame: 1920×1080px
- [ ] Set background color
- [ ] Enable Auto Layout (Vertical)
- [ ] Set Auto Layout mode to "Scale"
- [ ] Add padding (80px or 120px as specified)
- [ ] Add content elements with correct typography
- [ ] Set colors using brand palette
- [ ] Verify 60% white space minimum
- [ ] Name frame: "Master - [Template Name]"
- [ ] Test scaling behavior

## Notes

1. **Auto Layout "Scale" mode**: Ensures content scales proportionally when frame is resized
2. **White Space**: Maintain generous padding and spacing to achieve 60%+ white space
3. **Typography**: Use Scandia Medium for headlines, Scandia Regular for body text
4. **Colors**: Strictly use brand colors - no variations
5. **Naming**: All templates must start with "Master - " prefix
