## Critical Rules

### English

1. **Logo:** Always use official SVG files from `../assets/logos/`
2. **Colors:** Maximum 3 brand colors per composition; Orange = CTAs only
3. **Typography:** Scandia for text, Geist Mono for data/code only
4. **Text Case:** Always sentence case, never ALL CAPS (except acronyms)

---

### Español

1. **Logo:** Siempre usa archivos SVG oficiales desde `../assets/logos/`
2. **Colores:** Máximo 3 colores de marca por composición; Naranja = solo CTAs
3. **Tipografía:** Scandia para texto, Geist Mono solo para datos/código
4. **Caso de Texto:** Siempre sentence case, nunca MAYÚSCULAS (excepto acrónimos)
