## Template 1: Master - Title Slide

### Step 1: Create Frame
1. Create new frame: **1920×1080px**
2. Name it: **"Master - Title Slide"**
3. Set background: **#FFFFFF** (White)

### Step 2: Set Up Auto Layout
1. Select frame
2. Enable **Auto Layout** (Shift+A)
3. Set direction: **Vertical**
4. Set alignment: **Center**
5. Set padding: **120px** (all sides)
6. Set resizing: **Scale** (important!)

### Step 3: Add Logo Placeholder
1. Create rectangle: **200px width**, auto height
2. Add to Auto Layout
3. Set spacing below: **64px**
4. (Replace with actual logo later)

### Step 4: Add Headline
1. Create text layer
2. Font: **Scandia Medium**
3. Size: **100px**
4. Color: **#121212** (Black)
5. Letter spacing: **-1%**
6. Text: "AI Huevos" (placeholder)
7. Alignment: **Center**
8. Spacing below: **32px**

### Step 5: Add Tagline
1. Create text layer
2. Font: **Scandia Regular**
3. Size: **24px**
4. Color: **#121212** (Black)
5. Letter spacing: **-1%**
6. Text: "Tagline text" (placeholder)
7. Alignment: **Center**
8. Spacing below: **16px**

### Step 6: Add Subline (Optional)
1. Create text layer
2. Font: **Scandia Regular**
3. Size: **18px**
4. Color: **#9B9B9B** (Gray)
5. Letter spacing: **0%**
6. Text: "Subline text" (placeholder)
7. Alignment: **Center**

### Step 7: Verify
- [ ] Auto Layout set to "Scale"
- [ ] 60%+ white space
- [ ] All typography matches specs
- [ ] Frame named correctly

## Tips

1. **Auto Layout "Scale" mode**: This is critical! It ensures content scales proportionally when the frame is resized.

2. **White Space**: Use generous padding (80px-120px) and spacing between elements to achieve 60%+ white space.

3. **Typography**: If Scandia font isn't loaded, use a similar sans-serif as placeholder, then replace with Scandia.

4. **Testing**: After creating each template, test by resizing the frame to ensure "Scale" mode works correctly.

5. **Organization**: Consider creating a "Masters" page in your Figma file to keep all templates organized.
